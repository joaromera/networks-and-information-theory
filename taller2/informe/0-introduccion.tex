\section{Introducción}

El objetivo del presente trabajo es entender los protocolos involucrados a nivel de red con la herramienta \textit{traceroute}. Para esto, se implementó una versión basada en los mensajes \textit{echo request/reply} del protocolo ICMP y se la utilizó para analizar rutas a sitios web de universidades en diferentes continentes. En este análisis esperamos obtener información de cómo estas rutas se adaptan a los cambios topológicos gracias a los mensajes de control ICMP.
